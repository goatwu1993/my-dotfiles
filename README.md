## About

My dotfiles for macOS

## Step Up

```bash
brew install stow
# Stow only works if directory is right under HOME
cd ${HOME}

git clone https://github.com/goatwu1993/my-dotfiles
cd my-dotfiles
```

## Install

```bash
# Stow example
stow slate
```
